# FortiGate Parser Coverage

## Supported Formats
- [x] FortiOS Space-Separated Key-Value (KV) format

## Validated Scenarios
- [x] Traffic Forward (Accept/Deny)
- [x] System Events (Login/Cofig/Status)
- [x] Security Events (AntiVirus/IPS/WebFilter)
- [x] Quoted vs Unquoted Values
- [x] Missing Date/Time fields (Fallback to current time)
- [x] Syslog Header Prefix (optional)
- [x] Adversarial Key Injection
- [x] Null/Empty Values

## Tested FortiOS Versions
- FortiOS v5.x / v6.x / v7.x
